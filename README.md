# 🏠 Smart Home Dashboard 💡🌀🌡️

A modern Smart Home Dashboard developed using C# and ASP.NET MVC for monitoring and controlling smart home devices.

## 🚀 Features
- 💡 Light control system with ON/OFF status
- 🌀 Fan control management
- 🌡️ Environment monitoring with temperature display
- 🛏️ Room occupancy/status tracking
- 🔄 Sensor refresh functionality
- 📱 Clean and responsive dashboard UI
- 🏗️ MVC architecture for maintainable code structure

## 🛠 Technologies Used
- C#
- ASP.NET MVC
- HTML
- CSS
- JavaScript
